# Installation and Setup
To get the application running on your machine, follow the steps below:

## Clone the Repository:

```
git clone https://github.com/ZirixCZ/gin-htmx
```

## Navigate to the Repository's Directory:

```
cd gin-htmx
```

## Run the Application:
```
go run main.go
```
